# Hotkeys

## Switch view
Cycle window focus: `Mod+,` / `Mod+.`, i.e. `Mod+ <>`
Switch workspace: `Mod+Tab`
Focus left / right column: `Mod+H` / `Mod+L`
Focus top / bottom window: `Mod+K` / `Mod+J`
Focus workspace `1..9`: `Mod+[1..9]`
Move window to workspace `1..9`: `Mod+Shift+[1..9]`
Move columnn to workspace `1..9`: `Mod+Ctrl+[1..9]`

Run app: `Mod+R`
Open / focus browser `Mod+B`
Open / focus terminal `Mod+T`

## Resize view
Maximize column width: `Mod+F`
Fullscreen window: `Mod+Shift+F`
Decrease / increase columnt width: `Mod+Minus` / `Mod+Plus`
Decrease / increase window height: `Mod+Shift+Minus` / `Mod+Shift+Plus`
